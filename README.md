# 🌌 Sobre o Projeto

A **Cruzadinha Tech** é um jogo educativo com visual cyberpunk/neon criado para ensinar conceitos de tecnologia de forma divertida, moderna e interativa.

O projeto combina:

✨ Interface futurista  
🧠 Perguntas tecnológicas  
🎮 Interatividade  
⚡ Efeitos neon  
🚀 Experiência dinâmica
